# edu-adapt-api

Проект представляет собой серверную часть системы, предназначенную для хранения, поиска и управления учебными материалами, которые используются в процессе обучения иностранных студентов. 

Мы разрабатываем REST API, которое позволяет:
* **Загружать материалы** (книги, статьи, видео)
* **Искать** по тегам
* **Тегировать** контент для удобной фильтрации

## Стек

* **Язык:** Go 1.23
* **Web Framework:** Gin
* **База данных:** PostgreSQL 15
* **Инфраструктура:** Docker & Docker Compose
* **Документация:** Swagger (OpenAPI)

## Как запустить

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/glycek/edu-adapt-api.git](https://github.com/glycek/edu-adapt-api.git)
    cd edu-adapt-api
    ```

2.  **Настройте окружение:**
    ```bash
    cp .env.example .env
    ```

3.  **Запустите:**
    ```bash
    docker-compose up --build
    ```

Сервер запустится на `http://localhost:8080`

## Документация API

После запуска сервера документация доступна по адресу:
`http://localhost:8080/swagger/index.html`

Там можно протестировать все ручки (Endpoints) прямо из браузера.
